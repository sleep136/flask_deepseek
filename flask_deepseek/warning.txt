注意事项：
docker 代理 https://docker.1ms.run
依赖镜像 milvusdb/milvus:v2.2.9 、 quay.io/coreos/etcd:v2.0.0_rc.1

下面命令有问题建议使用docker compose 启动
milvus 启动命令：
docker run -d --name milvus -p 19530:19530 -p 9091:9091 milvusdb/milvus:v2.2.9 milvus run standalone
etcd 启动命令：
docker run -d  --name etcd-single-node -p 2379:2379 quay.io/coreos/etcd:v2.0.0_rc.1 /usr/local/bin/etcd --name etcd0 --advertise-client-urls http://0.0.0.0:2379 --listen-client-urls http://0.0.0.0:2379
